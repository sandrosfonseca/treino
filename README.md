# 💪 Plano de Treino Personalizado - Meta 67kg

Um aplicativo web completo para acompanhamento de treinos personalizados, desenvolvido para um homem de 43 anos com o objetivo de reduzir de 70kg para 67kg.

## 📋 Descrição

Este projeto oferece um plano de treino estruturado para 4 dias por semana, utilizando equipamentos básicos de academia em casa:
- Banco regulável
- Halteres (5kg, 10kg, 18kg)
- Barras paralelas (1m)
- Tatame 1x2,5m

## 🎯 Funcionalidades

### Principais
- ✅ **Separação de exercícios por dia** (Push, Pull, Legs)
- ✅ **Checkboxes interativos** para marcar conclusão dos exercícios
- ✅ **Persistência de dados** com localStorage
- ✅ **Barra de progresso** do peso corporal
- ✅ **Estatísticas em tempo real** (total, concluídos, taxa de conclusão)
- ✅ **Links para vídeos demonstrativos** no YouTube
- ✅ **Modo escuro** para conforto visual
- ✅ **Design responsivo** para mobile e desktop
- ✅ **Histórico de treinos** com datas
- ✅ **Timer de descanso** entre séries
- ✅ **Exportação em PDF** do progresso
- ✅ **Gráficos de evolução** do peso e desempenho

### Estrutura Semanal
| Dia | Tipo de Treino | Foco | Duração |
|-----|---------------|------|---------|
| Segunda | PUSH | Peito, Ombros, Tríceps | ~2,5h |
| Quarta | LEGS | Pernas, Glúteos, Core | ~2,5h |
| Sexta | PULL | Costas, Bíceps, Antebraços | ~2,5h |
